# React useRef Deep Dive

React `useRef` is one of the most powerful hooks.

It helps:
- persist values
- access DOM elements
- avoid unnecessary renders

---

## Why useRef matters

`useRef` does not trigger re-render.

```js
const ref = useRef(null);

console.log(ref);
```
